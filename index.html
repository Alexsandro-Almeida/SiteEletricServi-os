Projetos

Detalhes sobre os projetos realizados.

Projeto Subestação 225KVA

Um projeto de subestação de 225 kVA envolve a instalação de um transformador, cabinas de subestação, medidores, disjuntores, e sistemas de proteção. O processo inclui levantamento de dados, dimensionamento, projeto técnico, obtenção de licenças, construção, testes e manutenção. Esse tipo de subestação é usado para distribuir energia elétrica de forma eficiente e segura em áreas residenciais ou comerciais. A subestação garante o fornecimento contínuo e seguro de energia.

Projeto Eletrico SPDA Predial

Um projeto de Sistema de Proteção contra Descargas Atmosféricas (SPDA) predial envolve a instalação de para-raios, aterramentos e condutores para proteger edificações contra raios. O processo inclui levantamento de dados, dimensionamento, projeto técnico, obtenção de licenças, instalação e testes. Esse sistema é fundamental para garantir a segurança estrutural e elétrica do edifício, evitando danos e garantindo proteção contra surtos elétricos. A manutenção regular é essencial.

Projeto Sistema Solar Diagrama Unifilar

Um projeto de sistema solar com diagrama unifilar envolve a instalação de painéis solares, inversores e sistemas de armazenamento de energia para captar e converter energia solar em eletricidade. O processo inclui levantamento de dados, dimensionamento, elaboração do diagrama unifilar, obtenção de licenças, instalação e testes. Esse sistema é essencial para gerar energia renovável e sustentável, reduzindo a dependência de fontes não renováveis. A manutenção regular é crucial para o desempenho.

Concha acústica Prefeitura

Um projeto elétrico de Concha Acústica para uma prefeitura envolve a instalação de sistemas elétricos para suportar eventos culturais e esportivos. Isso inclui a instalação de iluminação adequada, sistemas de som e vídeo, além de cabos e conexões seguras para garantir a segurança e o conforto dos frequentadores. O projeto também pode incluir a implementação de sistemas de emergência, como alarmes e extintores, para garantir a segurança dos participantes em caso de emergência A manutenção regular é crucial.

Projeto Eletrico Comercial Escritorio de Advocacia

Um projeto para um escritório de advocacia comercial envolve a instalação de sistemas de iluminação eficiente, tomadas elétricas, e infraestrutura de rede para computadores e equipamentos de escritório. O projeto deve garantir a distribuição segura e adequada de energia elétrica, conforme normas técnicas e regulamentos locais. Além disso, inclui sistemas de segurança como iluminação de emergência, alarmes e detectores de fumaça para garantir proteção.
